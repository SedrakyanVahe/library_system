# Library Management system

#### This programm can help you manage your books

## Tools

* PyQt5
* XAMPP 8.1.1
* MySql

## Installing

#### git clone https://github.com/SedrakyanVahe/library_system.git

## Running

#### Run XAMPP
#### Open phpMyAdmin and do following commands from 'sql_commands.txt' file

#### cd library_system
#### python3 main.py
